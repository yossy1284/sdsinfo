新入生向けおすすめパソコンガイド

新入生向け パソコン
2026-03-16

SDS在学生の実感として、1〜2年次は高スペックPCがなくても困らないという声が多い。授業で重い処理が必要な場面では基本的にGoogle Colabなどのクラウド環境を使うため、ローカルのマシンパワーはそこまで必要ない。

とはいえ4年間使うことを見据えると、ある程度のスペックは確保しておきたい。この記事では、SDS新入生が選ぶべきスペックの目安と、具体的なおすすめモデルを紹介する。

スペックの目安

CPU

  • IntelでもRyzenでも可
  • 10万円前後のノートPCに搭載されているCPUは基本的に推奨スペックを満たしているため、特に気にしなくてOK
  • 生協PCのCPUは明らかにオーバースペック。NPU（AI処理用プロセッサ）も現状不要
  • CPUやストレージよりもメモリを優先した方が体感への影響が大きい

メモリ

  • 迷うなら16GB推奨
  • 8GBでもレポート作成やWeb閲覧は可能だが、ブラウザでタブをたくさん開いたり複数アプリを同時に立ち上げるだけでも足りなくなることがある
  • SDSの授業（プログラミング演習2、発展情報科目など）ではローカル開発環境を使うこともあり、16GBあると安心
  • 4年間の使用を見据えると16GBにしておく方が後々困らない

ストレージ

  • 512GB推奨
  • 256GBでも通常の大学用途なら問題なし（下記おすすめモデルは512GBが中心）
  • ゲームもやりたい人は512GB以上必須

「安く買って、必要になったら買い替える」戦略もアリ

8GB / 256GBなどの廉価モデルを買って1〜2年使い、必要になったら買い替えるという考え方もある。1〜2年のうちにAI事情も変わるし、Macなら下取りも出せる。そもそも履修方針次第では、情報系の発展科目やゼミを取らなければ高スペックが必要になる場面自体が少ない。逆にマシンパワーが求められるようなゼミほど、貸与PCやクラウドの計算機資源が用意されていることが多い。

購入時のポイント

値段

  • 10〜15万円がボリュームゾーン
  • この価格帯かつ大手メーカーの製品なら基本的に問題ない
  • 10万円以下でもDell 14やIdeaPadなど高コスパモデルは存在する

重さ

  • 軽ければ軽いほど良いが、軽さを売りにしている製品はデメリット（バッテリー持ち・端子数など）があることも
  • 2kgを超えると持ち運びがかなりつらいので、1kg台で探したい

納期

  • 新学期に間に合うかチェック必須
  • 記載の納期より遅れることもあるため、「即納」と書いてあると理想
  • 家電量販店なら即日入手可能だが割高。メーカー直販がおすすめ

OS（Windows / Mac）

  • どちらでも全く問題ない。SDSの発展科目を含め、OSの違いで困ることはない
  • AI系のツールやライブラリもMacで問題なく動作する
  • 好みや予算で選んでOK

セキュリティソフト

  • 市販のセキュリティソフトは不要。Windows標準搭載のWindowsセキュリティで十分
