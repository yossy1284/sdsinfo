## はじめに

本ガイドは一橋大学ソーシャル・データサイエンス学部（SDS）の履修に役立つ情報を、学生有志がまとめたものです。内容には編集部の主観が含まれます。正確な情報は必ず **CELSのシラバス** や **学士課程ガイドブック** で確認してください。

誤りを見つけた場合は [公式X (@SDS_info2024)](https://x.com/SDS_info2024) のDMまでお知らせください。

## 1年間の流れ

一橋大学は**4学期制**（春・夏・秋・冬）です。

| 月 | 内容 |
|---|---|
| 4〜5月 | **春学期**（春科目・春夏科目）→ 試験 |
| 6〜7月 | **夏学期**（夏科目）→ 試験 |
| 7月 | 夏期集中講義 |
| 8月 | 夏季授業休業 |
| 9〜10月 | **秋学期**（秋科目・秋冬科目）→ 試験 |
| 11〜12月 | **冬学期**（冬科目）→ 冬季授業休業 → 試験 |
| 1月 | 冬期集中講義 |
| 2〜3月 | 春季授業休業 |

- 通年科目は春学期〜秋学期にわたります。
- **集中講義**は夏期・冬期の授業休業期間に短期集中で単位修得を目指す科目です。

## 単位の数え方

各授業の単位数は、以下の計算で求められます。

> **ターム数 × 1週あたりのコマ数**

ターム数とは、春夏秋冬の各学期を1タームとして1年間は4タームで構成されます。

- 「春夏 月1」→ 2ターム × 週1コマ = **2単位**
- 「秋 火2金2」→ 1ターム × 週2コマ = **2単位**

※例外として「スポーツ方法」は上記の半分の単位になります（例: 春夏 週1コマ → 1単位）。

1単位あたりの総学修時間は45時間として設定されています。

### GPA制度

GPA（Grade Point Average）は個々の学生の学習到達度をはかる指標です。

- 履修した科目の成績を上記のGPで点数化し、1単位あたりの成績平均点を求めたもの
- GPAに算入されない科目もあります
- 奨学金や留学制度などでGPA数値が用いられます（一定以上のGPAが必要となる場合あり）
- 詳細は「学士課程GPA制度に関する要項」を参照してください

## CAP制限

**年間で取得できる単位の上限は44単位、各学期は上限14単位** です。

- 集中講義など、CAP制限の対象外となる授業もあります。
- GPA 3.8以上の場合、年間48単位・各学期16単位まで引き上げ可能です。

## 卒業までの流れ

1，2年次を **前期課程**、3，4年次を **後期課程** といいます。

- 前期課程から後期課程に進むことを **進学** といい、**進学要件** を満たす必要があります。
- 卒業するためには **卒業要件** を満たす必要があります。

> 履修計画 → 履修登録 → 授業 → 試験・レポート → 合格 → **単位修得**
> これをくりかえし、進学・卒業に必要な単位を揃えていきます。

## SDS学部のカリキュラム構造

### 授業科目の区分

授業科目は **全学共通教育科目** と **学部教育科目** に分けられます。

- **全学共通教育科目** — 外国語、数理・情報、運動文化、人文学の区分があり、全学部の学生が履修する科目群
- **学部教育科目** — 各学部が開講する科目で、導入・基礎・発展などの区分がある

SDS学部の学部教育科目は **導入 → 基礎 → 発展** と段階的に学びを積み上げていく構成です。

## 他学部教育科目

本学では、所属する学部以外の学部教育科目（他学部教育科目）も履修可能です。

- 履修できる学部や学年に指定がある科目があるので注意してください

## 1年生の履修選択

> CAP 44単位 ＝ 自動登録28単位 ＋ 社会科学入門8単位 ＋ 自由8単位

「SDS入門Ⅰ・Ⅱ」「線形代数Ⅰ・Ⅱ」「微分積分Ⅰ・Ⅱ」「統計」「情報リテラシー」「AI入門」「プログラミング基礎」「PACEⅠ・Ⅱ」の合計28単位が自動登録済みです。残り16単位のうち：

**残り8単位** — 編集部のおすすめはPACE以外の外国語科目を8単位取ること。進級条件に含まれるうえ抽選制のため、早めの確保が安心です。抽選で外れた場合は他学部科目もおすすめです（卒業条件に6単位必要）。

※1年生が履修できる他学部科目は学部導入科目のみです。
※外国語の組み合わせ例: 第2外国語「○○(総合)」で8単位 / 「○○(速習)」4単位＋英語2科目4単位 / 英語4科目で8単位

必修はゼミとPBL演習のみで自由度が高いですが、卒業要件として以下を確認しましょう：

- 「ビジネス・イノベーション科目」から2単位以上
- 「社会課題解決科目」から2単位以上
- データサイエンス科目のうち学部発展科目を2単位以上
- 他学部科目6単位が未履修なら早めの取得を

## 履修登録の流れ

履修登録には学務情報システム **CELS** を使います。

| 登録方法 | 説明 |
|---|---|
| **自動登録** | 大学が登録（指定必修科目） |
| **WEB抽選** | 学生が抽選に申し込み、当選すると登録される |
| **履修登録** | 学生が自分で登録 |

- コマの重複やCAP制限に注意してください
- 秋冬学期の履修見通しをもって時間割を作ることが大切です
- 履修登録がないと科目を受講できません

## SDS生が利用できない制度

以下の制度はSDS学部生は対象外です。

- グローバル・リーダーズ育成事業（グローバル・リーダーズ・プログラム）
- 学部・大学院修士課程5年一貫教育プログラム
- 国内単位互換制度（四大学連合「複合領域コース」等）
- 転学部

※教員免許状：社会学部・経済学部の教職課程を他学部受講として履修することは不可能ではないが、多くのCAPを使うため現実的には困難です。
